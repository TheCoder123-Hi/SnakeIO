# SnakeIO

A pixel-style Snake game made with Python's built-in `turtle` drawing library.

SnakeIO is a simple arcade game where you guide the snake around a blocky grid, eat food to grow longer, and avoid crashing into the walls or yourself. The game is drawn with square turtle graphics to give it a retro pixelated look.

## Features

- Pixelated grid-based gameplay
- Snake movement with arrow keys or WASD
- Food pickup and score tracking
- Best score display
- Game over screen with restart option
- Built using only Python and `turtle`

## How to Play

Move the snake around the board and eat the red food blocks. Each food block makes the snake longer and increases your score.

Avoid:

- Running into the wall
- Running into your own body

## Controls

| Key | Action |
| --- | --- |
| Arrow Keys | Move the snake |
| WASD | Move the snake |
| Space | Restart after game over |

## How to Run

Make sure Python is installed on your computer.

Then run:

```bash
python snake_game.py
```

If your computer uses `python3` instead of `python`, run:

```bash
python3 snake_game.py
```
